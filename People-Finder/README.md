# People-Finder

This is a website made to practice javascript skills. 
This website displays various people names, their selected images and bio of them in form of cards. You can search for a particular person with their name and there is a button to create a new card if you want.
For creating new card you can click on '+' button on the top left corner. A form will appear where you got to enter name, url of the desired picture and a short description about that person.
If you click on any card you can see two options displaying on the top right corner, they are - Delete and Edit. Delete deletes any user and edit helps you edit the existing information.
